Memory-Aware Medical RAG Assistant

A context-aware medical AI assistant leveraging RAG, hybrid memory, and LLMs.

View on GitHub

🚀 Features

  • Advanced Query Optimization: Multi-stage query rewriting, decomposition, alternative phrasing, and deduplication.
  • Hybrid Memory Architecture: Conversation, Entity, Episodic, and Vector Memory with long-term semantic memory.
  • Retrieval-Augmented Generation (RAG): Combines query rewriting, multi-query retrieval, cross-encoder re-ranking, and LLM responses.
  • Document Re-Ranking: Ensures the most relevant medical information is retrieved using cross-encoder models.
  • Streaming Responses: Real-time token streaming for web interfaces and chat applications.

💡 Example Workflow

User Question: What are the symptoms of kidney stones?

  1. Query rewriting generates optimized search queries
  2. Vector database retrieves relevant document chunks
  3. Documents are re-ranked using a cross-encoder model
  4. Context is combined with conversation memory
  5. LLM generates a medically structured response
