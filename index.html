Memory-Aware Medical RAG Assistant

A context-aware AI medical assistant combining Retrieval-Augmented Generation, hybrid memory architecture and intelligent query rewriting.

View Source Code Live Project Page

🚀 Core Features

Advanced Query Optimization

Conversation-aware query rewriting, decomposition and multi-query generation to improve retrieval recall.

Hybrid Memory Architecture

Combines conversation, entity, episodic and vector memory to maintain contextual awareness.

Retrieval-Augmented Generation

Integrates vector retrieval, context assembly and LLM generation for accurate medical responses.

Cross-Encoder Re-Ranking

Improves retrieval precision by re-ranking documents before passing them to the LLM.

Streaming LLM Responses

Tokens stream in real time for responsive conversational interfaces.

💡 Example Workflow

User Question: What are the symptoms of kidney stones?

  1. Query rewriting generates optimized search queries
  2. Vector database retrieves relevant medical documents
  3. Documents are re-ranked using a cross-encoder model
  4. Conversation context and memory are combined
  5. The LLM generates a structured medical answer
