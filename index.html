Memory-Aware Medical RAG Assistant

A context-aware medical AI assistant leveraging RAG, hybrid memory, and LLMs.

View Source Code Live Project Page

🚀 Features

  • Advanced Query Optimization Multi-stage query rewriting including conversation awareness, decomposition, multi-query generation, and deduplication.
  • Hybrid Memory Architecture Conversation, Entity, Episodic and Vector memory working together to maintain long-term context.
  • Retrieval-Augmented Generation (RAG) Combines vector retrieval, re-ranking and LLM generation for accurate medical answers.
  • Cross-Encoder Document Re-Ranking Improves answer relevance beyond standard vector similarity.
  • Streaming Responses Real-time token streaming for responsive chat interfaces.

💡 Example Workflow

User Question

What are the symptoms of kidney stones?
  1. Query rewriting generates optimized search queries
  2. Vector database retrieves relevant medical documents
  3. Documents are re-ranked using a cross-encoder model
  4. Context is combined with conversation memory
  5. The LLM generates a structured medical response
