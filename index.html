Medical RAG Assistant

Memory-Aware Medical RAG Assistant

A context-aware AI medical assistant combining Retrieval-Augmented Generation, hybrid memory architectures, and intelligent query rewriting to provide accurate and contextual medical information.

View Source Code on GitHub

🚀 Core Features

Advanced Query Optimization

Conversation-aware query rewriting, decomposition and multi-query generation improve retrieval recall.

Hybrid Memory System

Combines conversation memory, entity memory, episodic memory, and vector memory for contextual awareness.

Retrieval-Augmented Generation

Integrates external medical knowledge with LLM reasoning to produce accurate responses.

Cross-Encoder Re-Ranking

Ensures the most relevant documents are prioritized before passing to the LLM.

Long-Term Vector Memory

Stores important interactions using semantic embeddings for persistent knowledge.

Streaming LLM Responses

Tokens stream in real-time for a responsive conversational interface.

💡 System Workflow

1. User Query — User submits a medical question.
2. Memory Context Builder — Conversation and entity memory provide context.
3. Query Rewriting Pipeline — Optimized search queries are generated.
4. Vector Retrieval — Queries retrieve relevant medical documents from the vector DB.
5. Similarity Filtering — Irrelevant documents are removed.
6. Cross-Encoder Re-Ranking — Documents ranked by semantic relevance.
7. Context Construction — Top documents merged with memory context.
8. LLM Generation — Language model generates structured response.
9. Memory Update — Important interactions stored in long-term memory.
